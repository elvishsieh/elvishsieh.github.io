Open Source Code of PLM

  • OpenPLM is a product oriented open source PLM ( Product Lifecycle Management) software.
  • DocDokuPLM is a high end Open Source PLM solution. The project purpose is to develop a comprehensive, robust open source PLM (Product LifeCycle Management) solution.
  • OdooPLM is an Open Source module for Odoo that allows you to manage PLM Data inside
  • beCPG is an open source PLM software designed to manage the lifecycle of products, accelerate innovation and reduce time-to-market while improving product quality.

Open Source of Project Management

  • MyCollab is a multi-purpose solution which features tools for customer relationship management, document management, and project management.
  • Agilefant has focuses on agile approaches involving Kanban and Scrum, but also can be bent to support traditional approaches to project management such as the waterfall method.
  • LibrePlan is a web-based application, making project management available to not just the project manager, but the entire project team, and if necessary across organizations.

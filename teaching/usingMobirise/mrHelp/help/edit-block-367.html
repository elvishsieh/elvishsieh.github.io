區塊管理

在這個最後的文件中，我們學習到如何建立一個 Mobirise 專案，以及建置第一個網頁，現在，讓我們我們檢視細節而有能力來實作往後的網站

你網站的頁面可能包含多個區塊，為了效率及方便管理，在 Mobirise 的頁面當中，你能夠在任一地方輕鬆地拖拉、替換你想要的區塊

  1. 點選 Drag 按鈕並且按住滑鼠，再拖拉區塊 (請看圖例)
  2. 每個區塊可以依照你想要的來設定，實際上你有兩種選項，第一種是 “齒輪狀” Block Parameters，另一種是手動的設定，這種方式需要編輯網頁程式碼的知識
  3. 當頁面不需要區塊時，你可以簡單的刪除它
  4. 在 Save to User Block 按鈕，Mobirise 提供你儲存區塊的機制，把自己收集的區塊儲存起來，但是，僅在相同的主題中可以使用 使用者區塊

另一個重要的細節是檔頭及頁尾在所有頁面都是相同地，所以，在一個頁面中建置檔頭或頁尾，在專案的所有頁面都有相同的檔頭及頁尾

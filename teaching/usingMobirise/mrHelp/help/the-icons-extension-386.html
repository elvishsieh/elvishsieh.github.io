Icons 擴充套件

Icons 擴充套件可以幫你將小圖示加入到選單的選項、按鈕、取代在區塊中的圖片或者標誌

如何加入 Icons 擴充套件?

注意，若沒有加入 Icons 擴充套件，你也可以加入免費的小圖示，只要點選下拉式選單的選項、按鈕、區塊中的圖片、標誌等，然後選擇你要的小圖示

  1. 開啟 Themes & Extensions 清單
  2. 購買 Icons 擴充套件
  3. 依照底下訂購後的指引來安裝 Icon 擴充套件

如何使用小圖示來取代圖片或標誌:

如果你的區塊或主題支援這個套件，你可以用小圖示來取代圖片

  1. 點選圖片
  2. 點選 Insert icon 按鈕
  3. 在小圖示清單中選擇一個小圖示
  4. 點選 Insert icon 按鈕

如何在選項中加入小圖示:

  1. 點選選單項目
  2. 點選 Insert icon 按鈕
  3. 在小圖示清單中選擇一個小圖示
  4. 點選 Insert icon 按鈕

為了找到你想要使用的小圖示，可以使用 Search 欄位來尋找，在 Select icons 視窗中，你也可以設定小圖示的顏色及大小

If you use an icon instead of an image, you can also add a link to it.

F.A.Q.

Is it possible to add a third-party icon font to Icons 擴充套件?

Unfortunately, it isn't supported.

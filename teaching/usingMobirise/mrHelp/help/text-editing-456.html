文字編輯

讓我們學習更多的文字編輯

首先，你可以在任何一個區塊中編輯文字，但是你不能單一個字，需要在區塊中編輯文字片段，它可以是圖片標題文字、按鈕內的文字

所以，若你點選文字元件內的文字，你可以改變整個元件的樣式，而不是你點選的那個單字

如你所見，你可以改變文字元件的顏色、設定縮排，甚至建立你的樣式及使用其他文字元件

Mobirise 也提供你 Google 字型及上傳你自己的字型

如果你從文字元件中選擇一個或一些字，你可以設為粗體或斜體樣式及特別的文字中插入連結

在文字元件中，為了讓特別的文字或片語更有效果及更多樣式，你可以使用 Code Editor 或在你發佈網站之後手動設定樣式

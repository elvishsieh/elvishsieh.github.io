Code editor 擴充套件

如果你有撰寫網頁程式碼的技能，你可以找到Code editor 擴充套件來使用，它可以直接在 Mobirise 中編輯網頁的 HTML 與 CSS 程式碼，除此之外，它提供你個別編輯每一個區塊

  1. 購買擴充套件
  2. 選擇你想要編輯的區塊，以及點選 Edit code 按鈕
  3. 然後你可以看到這個區塊的 HTML 和 CSS 檔案，因此，現在你可以任意的編輯想要的區塊內容，當編輯完成，點選 Save and close 或 Cancel and close 來取消你變更的內容
  • 自 Mobirise v3.10.2 起，你仍然可以藉由 Block parameters 解開HTML 編輯器來編輯文件區塊和按鈕
  • 你可以按 Undo 所有變更

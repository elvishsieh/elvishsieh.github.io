匯入與匯出

讓我們談論一個重要的事情，就是你的專案，當然，如果你想要讓你的網站有更完整的功能，你需要知道更多事情

在 Mobirise 有 “專案” 和 “網站” 專有名詞，它們是不同的意思

請注意 Mobirise 指定一個專案的副檔名為 “.mobirise”，例如： projectname.mobirise

  1. 匯出專案，請開啟 漢堡式選單
  2. 點選 Sites 頁籤，找到你想會出的專案，點選藍色 齒輪狀 按鈕
  3. 再開啟的視窗中找到你需要的 Export site 選項
  4. 選擇你要儲存得路徑並匯出專案

限讚你已經匯出專案到你的硬碟，可以分享給你的朋友，再次注意， Mobirise 僅能夠匯出這類型的檔案，當你在 Mobirise 改變這些專案，它是不會影響發佈的網站

在 Mobirise 左邊選單，開啟匯出的專案：

  1. 到 Sites，建立一個新的專案，點選 Import Mobirise site
  2. 再調腦中找到專案檔案，匯入 Mobirise

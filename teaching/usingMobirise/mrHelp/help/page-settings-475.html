Code editor - 頁面設定

利用 Code editor 擴充套件，你可以網頁的程式碼，如 <head>, <body> 及線條

  1. 購買 Code Editor 擴充套件
  2. 到 Pages 頁籤
  3. 選取你要編輯的頁面，然後點選齒輪狀按鈕
  4. 你可以看到設定新頁面出現： 在 <head> 標籤裡面, <body> 標籤 以及 <!DOCTYPE>, <html> and <head> 標籤之前

你能夠在欄位中輸入正確的程式碼，當你網站預覽時，你可以經由控制台來檢查部分程式碼/p>

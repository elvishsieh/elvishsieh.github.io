# Install and Setup

## PowerShell installation
1.  PowerShell could be found from [Powershell Github URL](https://github.com/PowerShell/PowerShell)
2. Download PowerShell-7.x.x-win-x64.msi or PowerShell-7.x.x-win-x64.zip for Windows System
3. Install PowerShell into default directory or extract the .zip to another directory wheres you want 

## UV installation
>  UV is an extremely fast Python package and project manager which is more fast then the pip package management, was written in Rust language.

### Installing UV with PowerShell command as below steps

1. Before you install UV, you should setup UV's environment varlabel as following
   - UV_INSTALL_DIR environment variable
   - UV_PYTHON_INSTALL_DIR environment variable
   - For example, install UV in D:\uvx64\bin directory
    ```shell
    UV_INSTALL_DIR = D:\uvx64\bin
    UV_PYTHON_INSTALL_DIR = D:\uvx64\python
    PATH = D:\uvx64\bin;D:\uv_x64\python\cpython-3.14.0-windows-x86_64-none
    ```
2. Installing UV command on Windows
    ```shell
    powerShell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
    ```
3. Setting the **PATH** environment variable to execute UV, if UV install in D:\uvx64\bin directory
    ```shell
    PATH=%PAHT%;D:\uvx64\bin
    ```
4. Make sure UV can be executed on Windows
    ```shell
    uv --version
    ```
5. Install Python with UV
    ```shell
    uv python install
    ```
6. Setting the **PATH** environment variable to execute Python, if install Python 3.14.2 version
    ```shell
    PATH=%PAHT%;D:\uv_x64\python\cpython-3.14.2-windows-x86_64-none
    ```
> More information for UV Python package and project manager refer to the slides of [UV - another PM](https://elvishsieh.github.io/teaching/python/appendixA.html#/3)
## How to set up for Think Python Solutions book on Windows
1. First of all, you should download the textbook from [Think Python Solutions Github repository](https://github.com/AllenDowney/ThinkPythonSolutions)
2. Extracting it to a folder what you want.
    ```shell
    mkdir D:\pyProject
    unzip ThinkPythonSolutions-3.zip -d D:\pyProjects
    ```
    > 7z x ThinkPythonSolutions-3.zip -o D:\pyProjects, if you used 7zip to extract it.
3. Install Jupyter Notebook
    ```shell
    cd \D:\pyProjects\ThinkPythonSolutions-3
    uv init .
    uv add jupyter notebook
    uv add -r requirements.txt
    ```
4. Activate UV virtual environment
    ```shell
    cd D:\pyProjects\ThinkPythonSolutions-3
    .\.venv\Scripts\activate
    jupyter notebook & // or ipython
    ```

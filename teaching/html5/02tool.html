多國語言翻譯軟體

  • Lingoes 是一個多國語言的翻譯軟體，也提供中翻英、英翻中的功能，有 Win 32 位元(Bits) 與 Win 64 位元(Bits)兩種版本：
    • 32 位元：Lingoes 2.9.2
    • 64 位元：Lingoes 2.9.2 x64
  • 下載並安裝
    • 下載完成後，直接點選執行檔進行安裝，「連續按下一步」應該可以安裝成功
    • 安裝完成後，必須再下載繁體「中文/英文翻譯字典」安裝到 Lingoes 翻譯軟體，這樣才具有離線翻譯的功能
      • 中翻英字典(Vicon Chinese(T)-English Dictionary)
      • 英翻中字典(Vicon English-Chinese(T) Dictionary)

Lingoes 設定與使用

  • 在安裝中翻英字典、英翻中字典的最後面步驟時，請把 Options 選項中的 Add to "Index Groups" 與 Add to "Text Capture Groups" 這二個選項都勾選
  • 開啟 Dictionaries 的設定，將 Text Capture Groups List 中 "Vicon English-Chinese(T) Dictionary" 和 "Vicon Chinese-English(T) Dictionary" 往上移動到 List 的最上面
  • 按下 Ctrl + Alt + L 鍵開啟 mini Windows 視窗，按右上方的 logo 圖案，選取 "keep windows always on top" 選項
  • 常用的快速鍵
    • Ctrl + Alt + L 鍵： Open/Close mini Windows 視窗
    • Alt + L 鍵：Open/Close main Windows 視窗
    • Ctrl + Alt + K 鍵：收起/回復mini Windows 的翻譯內容

Example of Emmet(1/2)

  • ul>li.fragment*3
    • <ul>
               <li class="fragment"></li>
               <li class="fragment"></li>
               <li class="fragment"></li>
</ul>
  • ul 標籤底下(>)有 li 標籤， li 標籤類別(.)名稱是 fragment，總共有 3 個(*3)
  • > 表示"下一層"，在 ul 標籤內部含有 li 標籤
  • * 表示"乘以"，li.fragment*3 就是產生 3 個 li.fragment

Example of Emmet(2/2)

  • 我們想要建立一個 4x6 的表格(table)
    • thead 標籤： 1 個 tr(row), 4 個 td(column)
    • tbody 標籤： 4 個 tr(row), 4 個 td(column)
    • tfoot 標籤： 1 個 tr(row), 4 個 td(column)
    table>thead>tr>td*4^^tbody>(tr>td*4)*4^tfoot>tr>td*4
  • table>thead>tr>td*4 : table 標籤內有 thead 標籤，下一層有 tr 標籤，再一層有 td 標籤，而 td 標籤有 4 個(td*4)
  • ^ 表示"上一層"，^^tbody 表示 tbody 往上二層，，所以，與 thead 同一層
  • (tr>td*4)*4 : 1 個 tr 內有 4 個 td，這樣的組合總共有 4 組[(1 個 tr 內有 4 個 td)*4)]，就等同於 4 個 tr，每個 tr 內都有 4 td

FileZilla FTP 軟體介紹

  • FTP 就是 File Transfer Protocol 的縮寫，檔案傳輸協定的意思
  • 你覺得使用瀏覽器來下載檔案有什麼問題？
    例如：台大的 FTP 伺服器
  • 問題在於：一個一個檔案下載很不方便
  • FileZilla有提供FTP Client/Server 的軟體
  • FileZilla Server 軟體可以架設 FTP Server，我們僅用 Client 軟體，將網頁上傳到 Web 伺服器

FileZilla 的設定

  • 主機 IP： 140.120.63.48
  • 使用者帳號與密碼：兩者都相同，助教會給你們一組編號
  • 使用預設的 21 Port，所以不需要特別做設定
  • 可以使用「快速連線(Quickconnect)」或者設定好「站台管理者(Site Manager)」來連線
  • 安裝及設定步驟相當簡單，所以不提供「可攜式」的 FileZilla 軟體，請自己嘗試看看
  • 站台管理者(Site Manager)設定：
    • File >> Site Manager >> New Site >> 輸入：主機名稱、帳號、密碼

網頁上傳的注意事項

  • 必須在自己的 home 目錄中建立一個 www 目錄
  • 網頁的首頁檔名可以是 index.php, index.html
  • 網頁請使用 SSL 協定，即 https://
  • 如果上傳成功，則可以輸入 「https://140.120.63.48/~你的編號」來檢查
  • 這台機器是 FreeBSD 的作業系統，請注意英文字母大、小寫有區別，不可混淆，檔名建議使用英文來命名

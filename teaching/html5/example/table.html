HTML5 所定義的表格標籤
Tag Description
<table> 定義一個表格
<th> 表格的檔頭
<tr> 表格的列
<td> 表格列內的一格
<caption> 表格的標題文字
<thead> 設定哪一群標籤是屬於表格的檔頭
<tbody> 設定哪一群標籤是屬於表格的主體
<tfoot> 設定哪一群標籤是屬於表格的表尾

表格的第一個例子
Month Savings
January $100
February $200
March $300
April $400

HTML 程式碼 

<table class="example">
<caption>表格的第一個例子</caption>
<tr>
	<th>Month</th>
	<th>Savings</th>
</tr>
<tr>
	<td>January</td>
	<td>$100</td>
</tr>
<tr>
	<td>February</td>
	<td>$200</td>
</tr>
<tr>
	<td>March</td>
	<td>$300</td>
</tr>
<tr>
	<td>April</td>
	<td>$400</td>
</tr>
</table>

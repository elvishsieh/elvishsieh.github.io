respimage

respimage polyfill(填空期=空窗期)

head 標籤內的部分原始程式碼：

<head>
    <script src="https://afarkas.github.io/respimage/respimage.js" async=""></script>
</head>

程式碼說明：

  • 直接連結到 respimage.js 套件存放在 GitHub 網站的程式碼
  • 注意： 不管是 CDN 還是這種方式，一定要能連上網路，如果你想樣離線開發，就必須要下載 .js 檔案
  • async : respimage.js 套件處理 DOM(Document Object Model) 時採用非同步的方式來處理，就是不需要等待 respimage.js，先處理別的事情，等 respimage.js 完成後再才處理 respimage.js 的要求

body 標籤內的部分原始程式碼：

<picture>
    
	<source  srcset="http://placehold.it/525x225 1.5x,
        	    http://placehold.it/350x150 1x"
			    media="(max-width: 380px)" />
	<source	srcset="http://placehold.it/1400x600/e8117f/fff 1.5x,
        	    http://placehold.it/1024x439/e8117f/fff 1x"
			    media="(max-width: 1050px)" />
	<img	srcset="http://placehold.it/2100x900/117fe8/fff 1.5x,
            http://placehold.it/1400x600/117fe8/fff 1x"
            src="http://placehold.it/420x180/117fe8/fff"
			alt="image with artdirection" />
</picture>

程式碼說明：

  • 當寬度在 380px 以下[media="(max-width: 380px)"] 顯示 http://placehold.it/350x150 的圖形；但是若解析度(放大)是 1.5 倍時，就顯示 525x225 圖
  • 相同地，當寬度在 1050px 以下就顯示 http://placehold.it/1024x439/e8117f/fff 的圖形；但是若解析度(放大)是 1.5 倍時，就顯示 1400x600 圖
  • 不在上面的範圍內時，就顯示 http://placehold.it/1400x600/117fe8/fff 圖

placehold.it 網站圖檔的使用及說明：

  • http://placehold.it/ 是提供任何大小圖檔作為網頁設計者測試的網站，是 imgix 所擁有及維護
  • 基本的用法：http://placehold.it/寬x高/背景顏色/文字顏色
  • 例如：http://placehold.it/1400x600/117fe8/fff
  • 說明： 寬x高 ‐ 1400x600，背景顏色‐ e8117f，文字顏色 ‐ fff(白色)

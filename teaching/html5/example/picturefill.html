head標籤內的部分原始程式碼：

<head>
    <!-- 使用 jsDelivr JavaScript CDN 來連結 picturefill.js 套件 -->
    <script src="https://cdn.jsdelivr.net/picturefill/3.0.2/picturefill.min.js"></script
</head>

程式碼說明：使用 CDN 方式將 picturefill.min.js 套件匯入到這個網頁，同學也可以下載 picturefill.min.js 套件來使用

body標籤內的部分原始程式碼：

<img sizes="(min-width: 40em) 80vw, 100vw"
    srcset="http://placehold.it/375x175/88e2a9/fff 375w,
    http://placehold.it/480x175/e8117f/fff 480w,
    http://placehold.it/768x175/117fe8/fff 768w" alt="…">

程式碼說明：

  • img 標籤是插入圖檔，W3C 標準已經制定完成，但是只有 src 的屬性，尚未正式公布 srcset, sizes 這兩個屬性的設定，所以需要 picturefill.js 套件來暫時替代
  • sizes 屬性： 設定 Media, Viewport 的寬度
  • srcset 屬性：當符合 sizes 設定的寬度時，會對應的擷取不同大小的圖檔
  • vw：是 viewport 的長度單位，所謂 viewport 是站在使用者觀看角度時的長度
  • w：width 寬度的意思

vw 是 Viewport 的縮寫，是 W3C 制定的一種相對的長度單位，標準的文件可以參考 Viewport-percentage lengths ，下面是一個換算的範例：

h1 {
   font-size: 8vw
}

程式碼說明： 如果 viewport 的寬度是 200mm, 則 h1 字體的 16mm (i.e. (8×200mm)/100).

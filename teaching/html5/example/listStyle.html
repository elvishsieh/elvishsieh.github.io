請選擇您喜歡的項目符號樣式

@counter-style blacknwhite {
  system: cyclic;
  symbols: ◆ ◇;
  suffix: " ";
}

ul {
  list-style: blacknwhite;
}
@counter-style 的範例是源自於 Mozilla CSS @counter-style Demo
更多的 @counter-style 資訊可以在 Mozilla Developer Network查到
注意： @counter-style 目前僅在 Firefox 33+ 中顯示
使用方法：
  • 複製上面的 CSS 程式碼到 style 標籤(internal)或者 CSS(external) 檔案
  • 在 body 中建立 ul 與 li 標籤

Bootstrap 的安裝

  • Bootstrap 有下列安裝方式 ：
    1. Bootstrap CDN:
    2. Bootstrap: 事先已經編譯好的套件
    3. Source code: 從 Less 原始程式碼來安裝
    4. Sass: 從 Sass 原始程式碼來安裝
  • 一般的使用者請採用第二種 Precompiled Bootstrap 來方式安裝 Bootstrap 套件
  • 在網站中下載下來的 Precompiled Bootstrap 套件沒有樣版、主題的範例，請到 Bootstrap Gatting started 的 Examples 中下載包含原始程式碼的完整套件
  • Webstie.zip 已經將這些安裝的步驟及相關的連結都處理好，請執行 installMenu.bat 批次檔即可

Grid System of Bootstrap

  • Grid System 把它想像成是一個無框方格
  • 每一個 grid 由 12 columns(行)組成一個 row(列)，換句話說，就是要湊成 12 columns
  • Bootstrap 規定每一個 grid system 包覆在 row 類別中，而 row 又不須被 container(or container-fluid)所包覆
    • <div class="container">
     <div class="row">
      <div class="col-md-8">.col-md-8</div>
      <div class="col-md-4">.col-md-4</div>
    </div>
</div>

Documentation of Bootstrap

  • git clone bootstrap:
    git clone https://github.com/twbs/bootstrap
  • Download and install Powershell
  • Installing Chocolatey: Open an administrative command prompt and paste the text from the chocolatey website
  • Install Ruby via Chocolatey: choco install ruby -y
  • Reopen a command prompt and install Jekyll:
    gem install jekyll
  • Install bundler: gem install bundler
  • From the root /bootstrap directory, run bundle install and
    bundle exec jekyll serve
  • Open http://localhost:9001 in your browser

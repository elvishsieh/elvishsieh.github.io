Content Delivery Network

  • CDN: Content Delivery Network 的縮寫，中文翻譯：內容傳遞網路。
  • CDN 網路所有者在世界各地建置許多內容完全相同的網站，讓使用者可以依照所在的位置，就近存取到幾乎完全一樣的網站內容。
  • Google CDN
  • Microsoft Ajax Content Delivery Network
  • cdnjs.com: 提供 Open Source Javascript, CSS 的 CDN
  • jsDelivr JavaScript CDN

Git 版本控制系統

  • 在撰寫文件、程式時，可能需要版本控制系統來管理你所寫的文件或程式碼，以便紀錄不同版本之間的差異及控制版本的內容。
  • 網路上有許多版本控制系統，大都採開放原始程式碼方式發行，較流行的版本控制系統有：Git, SVN(Subversion), Mercurial
  • Git 版本控制系統是 Linux 創始者 Linus Torvalds 所提出，對於開放原始程式碼的另一項重大的貢獻。
  • 我們使用 isocpp.org 成員 STL 編譯好的 with git 檔案
  • C++ IDE，初學者建議先使用 CodeBlocks，未來可以使用 Qt or Visual Studio

Node.js 與套件管理系統

  • Node.js® is a JavaScript runtime built on Chrome's V8 JavaScript engine.
    • 不需要 Browser 就可以開發、執行、測試 Javascript 程式
  • npm is the package manager for JavaScript.
    • 安裝某一個套件時，會將與這個套件相關的版本、套件一起安裝，不需要自己手動的安裝
    • 安裝 Node.js 之後，npm 就一起安裝
  • bower: 另一種 JavaScript 的套件管理軟體
    • 安裝： npm install -g bower
    • 指令說明： 使用 npm 安裝 bower 軟體
    • 指令說明： -g 表示 Global 全面性，就是安裝到這個系統，而不是在目前的目錄底下

編譯 Less 原始程式碼

  • 安裝 less ： npm install -g less
  • 撰寫 less 原始程式碼：(副檔名： .less)
    • @nice-blue: #5B83AD;
@light-blue: @nice-blue + #111;
#header {
  color: @light-blue;
}
#footer{
    color: @light-blue;
}
  • 編譯 less 原始程式碼：
    • 範例： lessc style.less style.css
    #header {
  color: #6c94be;
}
#footer {
  color: #6c94be;
}

Less 開發環境安裝與建置

  • 切換到專案的目錄： cd /d d:\projects
  • 安裝必要的軟體套件：
    • 安裝 gulp or grunt
      • npm install gulp
      • npm install gulp-less
  • 在專案的目錄(projects)中建立 一個 gulpfile.js 檔
    • var gulp  = require('gulp');  
var less  = require('gulp-less');

gulp.task('default', function() {  
  gulp.src('./src/lessTesting.less')
    .pipe(less())
    .pipe(gulp.dest('./css/'));
});
  • 執行 gulp: gulp

Installing Bootstrap from Source Code

  • Installing git: 請參考 Installing and Using Git
  • 安裝 Node.js
  • Update npm packages: npm update -g
  • Install bower: npm install -g bower
  • Install grunt: npm install -g grunt-cli
  • 切換到 projects 目錄： cd /d d:\projects
  • 安裝 bootstrap： bower install bootstrap
  • 切換到 bootstrap 並安裝相關的檔案：
    cd bower_components\bootstrap & npm install
  • Build Bootstrap： grunt
  • 完成上面的步驟，在 dist 目錄就會產生 Bootstrap 的 css, fonts, js 檔案

Gnu Wget

  • GNU Wget : a tool of mirror a website
  • Download for Windows:
    • download information on the Wget Wgiki
    • Windows binaries of GNU Wget
  • Mirror a website: wget -m URL
  • Example: wget -m http://v3.bootcss.com/
  • 再輸入上面範例的指令時，請特別注意：「環境變數」的這個問題，換句話說，就是要將目錄切換到 wget 解壓縮的目錄底下，才能輸入上面的指令

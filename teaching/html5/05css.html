階層式樣式表(CSS)

  • CSS 的英文全名是：Cascading Style Sheet，中文翻譯成：階層式樣式表
  • 問題是：什麼是 CSS ？
  • 我們說：每一個標籤都是某一種特定功能的標籤，像 <h1>標籤會將字體加粗、加大
  • 這些標籤內的字體、字體大小、顏色等等可以改變嗎？
  • 答案是：可以的，CSS 就是改變標籤屬性、屬性值的一種語法
  • CSS 是專有名詞，當同學了解這個專有名詞的意義以後，希望同學能慢慢地把「階層式樣式表」這個中文翻譯的詞忘掉

標籤的屬性與屬性值(1/2)

  • 每個不同標籤有不同的功用，也有不同的屬性及屬性值
  • 例如：編號第 2 列 <html lang="zh-Hant-TW"> 中， lang 是 <html> 標籤的屬性， zh-Hant-TW 是 <html> 標籤 lang 屬性的屬性值，
  • 特別注意： 標籤字母與屬性之間必須空至少一個空格
<!DOCTYPE html>
<html lang="zh-Hant-TW">
<head>
    <meta charset="UTF-8" />
    <title>頁籤上的標題</title>
</head>
<body>
      
</body>
</html>

標籤的屬性與屬性值(2/2)

  • 第一種格式：屬性="屬性值1" 屬性="屬性值2"
    • 屬性="屬性值1" 與屬性="屬性值2" 中間也必須要空格
    • 例如： <hr width="90%" color="red" />，屬性與屬性之間是空格，不可以接連著寫在一起，若接連著寫在一起，這樣瀏覽器無法正確解析第 2 個屬性與屬性值。
  • 第二種格式：style="屬性: 屬性值1; 屬性: 屬性值2"，
    • 這種格式是使用 style 這個保留字，配合 = 號並將屬性與屬性值放置在 "" 號內，屬性與屬性值中間以 : 號做區隔，以 ; 號作為結束符號。
    • 例如： <hr style="width: 90%; color: red;" />， 屬性與屬性值之間是 : 號，以 ; 號作為結束。
    • 特別注意：不管是 : 號或者是 ; 號，都是半形的英文

CSS 的寫法

  • Inline(內含)：就是標籤內的屬性與屬性值
  • Internal(內部)：使用 style 標籤
    • <head>
    <style>
        p{
            /*  這是備註的符號，與  */
            font-family: "Times New Roman", "標楷體";
            font-size: 16pt;
            line-height: 1.7em;
        }
    </style>
</head>
  • External(外部)：連結到外部的 css 檔案
    • <link rel="stylesheet" href="./css/style.css">

Selector

  • Type Selector
    • p{
    font-size: 16px;
}
  • ID Selector
    • #cat{
    color: gray;}
  • Class Selector
    • .dog{
    margin: 10px 5px 10px 5px /* 上 右 下 左 */
    padding 10px 5px 10px 5px /* 上 右 下 左 */}
p.box{
    margin: 10px 5px 10px 5px /* 上 右 下 左 */}

CSS Priority(優先順序)

  1. !important
    2. a {
    background-color: #555 !important;}
  2. Embedding the Styles
    • Inline Styles
    • Internal Styles
    • External Styles
  3. Selectors
    • ID Selector
    • Class Selector
    • Type Selector
  4. Default Browser Styles
